  curl -i -X POST https://<host>/mcp

         scheme "exact", network "eip155:196", amount "10000" = 0.01 USDT at 6 decimals). POST /mcp/ returns byte-identical 402.
  NOT gated: "/healthz", "/", "/badge/*", and the MCP handshake (initialize, tools/list).
