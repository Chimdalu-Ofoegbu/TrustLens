```python
UA = "TrustLens/1.0"
CACHE_DIR = Path("data/cache")     # already gitignored
RATE_S = 1.1                        # > 1 req/sec: sleep BETWEEN network fetches
TIMEOUT_S = 15.0
BASE = "https://www.okx.ai"
log = logging.getLogger("indexer.scraper")
```
Use `from __future__ import annotations`. Imports: `hashlib, json, logging, time`, `from pathlib import Path`, `import httpx`, `from bs4 import BeautifulSoup`, and from the indexer package `AgentRecord`, `name_key`, `derive_category`.

`_cache_path(url) -> Path`: `CACHE_DIR / (hashlib.sha256(url.encode()).hexdigest() + ".html")`. sha256-of-URL filename is the ONLY path component — no user-controlled segment (V12: no traversal).

`fetch(client, url, *, _pacer)` -> `str | None` (VERBATIM control flow from Pattern 1):
  - cache hit (`p.is_file()`) -> return `p.read_text(encoding="utf-8")` with NO sleep, NO network.
  - politeness: if a prior network fetch happened this run, `time.sleep(RATE_S)` BEFORE the request (pass a small mutable pacer, e.g. a `list`/counter, so the first fetch doesn't sleep).
  - `try: r = client.get(url, timeout=TIMEOUT_S)` — SINGLE attempt, NO retries. `except httpx.HTTPError as exc: log.warning("fetch failed url=%r: %s", url, exc.__class__.__name__); return None`.
  - `if r.status_code != 200: log.warning("fetch non-200 url=%r status=%d", url, r.status_code); return None` (covers 403/5xx).
  - on 200: `CACHE_DIR.mkdir(parents=True, exist_ok=True)`, write `r.text` (encoding="utf-8"), return it.

`parse_appstate(html, url)` -> `AgentRecord | None` (VERBATIM from Pattern 2, using the field mapping in <scrape_target>):
  - `soup.find("script", attrs={"id": "appState", "type": "application/json"})`; if `None` or no `.string` -> `log.warning("no appState script url=%r (SPA/markup change)", url); return None`.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| okx.ai HTTP response → indexer/scraper.py | Untrusted third-party marketplace content (the `appState` JSON is attacker-influenceable) crosses into our parser |
| scraped AgentRecord → indexer/db.py persist | Parsed values cross into SQLite (parameterized `?` writes) and into scoring |
| scraper file cache → data/cache/ filesystem | Response bodies written to disk keyed by URL hash |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering | parse_appstate field casts | mitigate | Every field cast (`float()`/`int()`) wrapped in try/except → None+WARNING on bad value; a hostile/oversized numeric string yields a soft parse-miss, never a crash or a poisoned record (V5). |
| T-05-02 | Tampering | scraped `category` → scoring reason text | mitigate | Option B: scraper never overwrites `category`; category_source stays "derived"; scoring/components.py already refuses non-canonical category text. Raw okx.ai codes can never reach a reason string. |
| T-05-03 | Tampering | cache filename from URL | mitigate | Cache path = `data/cache/` + `sha256(url).hexdigest() + ".html"` — no user-controlled path segment; no traversal (V12). |
| T-05-04 | Tampering / Info-leak | scraped CJK/newline text in logs | mitigate | All degradation logs use `%r` for url/id/name (log-injection + cp1252-console safe, per census convention); warnings carry url/status only, never raw body (V7). |
| T-05-05 | Denial of Service | hammering okx.ai (impoliteness → IP block) | mitigate | ≤1 req/s `time.sleep(1.1)` sleep-between + single attempt (no retry loop) + on-disk cache (cache hit = zero network) + hard 15s timeout (V12). |
| T-05-06 | Denial of Service | scrape failure breaks offline determinism | mitigate | `scrape_agents` catches everything and returns `[]`; refresh's 0/1/2 exit contract is driven only by census-read (1) and DB (2) stages; proven by the 403-MockTransport exit-0 test (Pitfall 1). |
| T-05-07 | Elevation / SSRF | detail URL construction | mitigate | URLs hardcoded to `https://www.okx.ai/agents/<id>` from a fixed `DEMO_AGENT_IDS` set (not user input); id constrained to `^\d+$` if ever non-literal (V13). |
| T-05-08 | Info Disclosure | scraper reachable from request path | mitigate | Scraper is offline-CLI-only; test asserts no `server/*` module imports `indexer.scraper` (V4, Pitfall 7). |
| T-05-09 | Denial of Service | resource exhaustion via giant response | accept | 15s hard timeout + single attempt bound the hang; httpx buffers the body but at okx.ai's ~50-100KB page sizes and a bounded demo set the memory cost is negligible. No streaming-size cap added (out of timebox; documented). |
